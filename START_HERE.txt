ピカッと！おばけミラー

起動:
1. npm install
2. npm run dev
3. ブラウザで表示

確認:
- npm run test:quality
- npm run lint
- npm run build

ゲーム内容:
- 鏡クリック回転のみ
- 1〜4人
- 3ラウンド固定
- EASY / NORMAL / HARD
- 色ライトを同じ色のゴールへつなぐ
- NORMAL/HARDは複数ライト
- HARDは分岐候補あり
- ResultはGameManager.ts経由でReact側へ返す

構成:
