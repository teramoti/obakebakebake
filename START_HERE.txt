ピカッと！おばけミラー

概要:
鏡をクリックで回転させ、ライトの光を同じ色のゴールへつなぐ交代制パズルミニゲームです。

起動:
1. npm install
2. npm run dev

確認:
- npm run test:quality
- npm run lint
- npm run build

主な構成:
- src/app/App.tsx
- src/app/screens/StartScreen/StartScreen.tsx
- src/app/screens/GameScreen/GameScreen.tsx
- src/app/screens/ResultScreen/ResultScreen.tsx
- src/game/GameManager.ts
- src/game/scenes/MirrorPartyScene.js
- src/game/systems/

仕様:
- 1〜4人対応
- EASY / NORMAL / HARD
- 3ラウンド固定
- 操作は鏡クリック回転のみ
- NORMAL/HARDは複数ライトと色ゴールあり
- HARDは分岐・固定ミラーあり
- FINISHで手動終了
- ResultはGameManager.tsからReact側へ返却

提出ZIPに含めないもの:
- node_modules
- dist
- public
- svg
